# Ответы на ключевые вопросы

## Проблема

Организация нейронной сети для выявления сложных нелинейных зависимостей и риска заболевания.

## Актуальность

Раннее и точное прогнозирование риска заболеваний позволяет своевременно организовать профилактические мероприятия. Современные методы на основе нейронных сетей демонстрируют высокую точность и адаптивность при работе с многомерными медицинскими данными [1, 6]. Нейросетевые подходы показывают потенциал в повышении точности прогнозирования, что делает их актуальными для внедрения в системы поддержки принятия врачебных решений [2, 5].

## Объект исследования

Процесс прогнозирования риска развития заболеваний на основе многомерных медицинских данных пациентов.

## Предмет исследования

Уровень точности прогнозирования риска заболевания, достигаемый с помощью нейросетевых моделей.

## Цель дипломного исследования

Разработать нейросетевую модель для прогнозирования риска заболеваний на основе многомерных клинических данных, обеспечивающую AUC-ROC не менее 90%.

## Цель на текущий семестр

Разработать прототип нейросетевой модели для прогнозирования риска заболеваний на основе многомерных клинических данных.

## Задачи

1. Выполнить обзор нейросетевых и ML-методов прогнозирования риска заболеваний [1, 5].
2. Спроектировать архитектуру нейросети с учётом особенностей многомерных данных [2, 5].
3. Обучить модель, подобрать гиперпараметры, провести оценку по метрикам (AUC-ROC, точность, полнота, F1) [1, 6].
4. Сравнить полученную модель с классическими алгоритмами (логистическая регрессия, деревья решений, бустинг) и проанализировать результаты [2, 6].
5. Исследовать интерпретируемость модели с помощью SHAP.
